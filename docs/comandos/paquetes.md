## Comandos necesarios

- Instalar nest globalmente
```
pnpm add @nestjs/cli
```
- crear un proyecto 
```
nest new nombre-proyecto
```
- Instalar Swagger
```
pnpm add @nestjs/swagger
```
- Instalar config para las variables de entorno
```
pnpm add @nestjs/config
```
- Instalar typeorm y pg
```
pnpm add @nestjs/typeorm typeorm mysql2
```
- Instalar dotenv
```
    pnpm add dotenv
```
- Instalar class validator
```
pnpm add class-validator class-transformer
```
- Instalar bcrypt para cifrar
```
 pnpm add bcrypt
```

-Para JWT
```
    pnpm add @nestjs/jwt
```
